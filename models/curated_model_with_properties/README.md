# MPTA-Repair Curated Dataset

This directory contains the curated benchmark for **MPTA-Repair**: Multi-Property Timed Automata Repair. It is generated from `models/model_with_ref` by `scripts/build_curated_dataset.py`.

Each model has at most three selected properties. Reference documents are intentionally not copied here; they remain in `models/model_with_ref`.

Model versions: 13
Total properties: 34

See `PROPERTY_SELECTION.md` for the meaning of each selected property and the selection rationale.
